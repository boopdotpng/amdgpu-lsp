# AMDGPU Language Server VS Code Extension

This VS Code extension launches the `amdgpu-lsp` server over stdio and provides completion, signature help, and documentation hovers for AMD RDNA/CDNA instructions.

## Features

- Completion and suggestions for instructions.
- Signature help while you type.
- Hover documentation from the ISA data.
